Jenkins plugin for RapidDeploy

This Jenkins plugin allows to invoke operations in your RapidDeploy server from Jenkins by defining a post-build action. The plugin can be used with RapidDeploy 3.3.3+.

You can find the sources here.

Versions

Version 1.0 (July, 2013)

Post build actions:

RapidDeploy project deploy: invoke a deployment task in RapidDeploy server.

Attributes:

  • RapidDeploy server url
  • RapidDeploy authentication token
  • RapidDeploy project
  • Target server name
  • Target environment name
  • Target instance name
  • Target application name
  • Deployment package name
